# MusiNexus Visual Style (MVP)

## Goal
A modern, music-first look: dark-friendly, vibrant gradient background, high contrast typography, soft glassy surfaces, big CTAs.

## Brand feel
- “Night club / concert” energy
- Premium but simple
- Fast and readable

## Layout
- Max width: 1200px
- Hero split layout (text left, illustration right)
- Rounded corners: 16–24px
- Shadows: soft, not harsh
- Glass panels for feature strip/cards

## Color tokens
- Background base: deep navy / indigo
- Primary: electric blue
- Accent: purple/pink glow (for gradients only)
- Text: near-white on dark, near-black on light
- Borders: subtle, 10–15% opacity

## Components
### Buttons
- Primary: solid blue, white text, rounded-lg, strong hover
- Secondary: dark surface with border
- Size: large in hero, normal elsewhere

### Cards / Panels
- Glass: background with opacity + blur
- Border: 1px translucent
- Padding: 16–20px
- Icons: simple line icons

### Typography
- H1: 44–56px desktop, 32–40px mobile, font-semibold
- Body: 16–18px, relaxed line-height
- Muted text: 70–80% opacity

## Theme (light/dark/system)
- Default: system
- Dark is first-class
- No flash: apply theme early

## Navigation
- Minimal top bar
- Left: logo
- Right: links + theme toggle + auth actions
